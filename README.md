## ğŸ› ï¸ Installation

Build `LiviaAI` from source and install dependencies:

1. **Clone the repository**  
   ```bash
   git clone https://github.com/bagaswijaya97/LiviaAI
   cd LiviaAI
   ```

2. **Restore NuGet packages**  
   ```bash
   dotnet restore
   ```

3. **Build the project**  
   ```bash
   dotnet build
   ```

4. **Run the application**  
   ```bash
   dotnet run
   ```
